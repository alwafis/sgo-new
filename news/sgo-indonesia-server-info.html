Stargate Online Indonesia

Informasi server dan introduction


Posted by Admin - 23 Juli 2022

Close Beta

Stargate Online Indonesia sudah di depan mata, nantikan update terkait tanggal close beta kami di website ini. Beberapa bulan kebelakang tim developer stargate online indonesia telah mempersiapkan client dan server stargate online dalam bahasa Indonesia, kami berharap dengan dibukanya server ini dapat membangkitkan nostalgia para player yang pernah memainkan game ini di server ofisial maupun yang baru memainkan game ini.

Server akan memiliki detail sebagai berikut:

1.5x exp rate

1.5x drop rate


Tim developer sangat berhati-hati dalam menentukan drop rate dan exp rate, beberapa pertimbangan terkait exp dan drop rate di atas adalah dari kapasitas development tim developer, sistem, dan perhitungan level cap dengan exp rate.

Kami membuka kritik dan saran terkait hal ini, yang mana dapat disampaikan melalui channel discord kami ataupun melalui social media kami yang tertera pada web kami. Tim developer mengucapkan terima kasih kepada para player yang sudah bersabar menunggu sampai server ini dibuka.


Enjoy the game and see you soon at Nilia Capital!

~Stargateonline.id maintainer
